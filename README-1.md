# ✦ Lyra AI — Intelligence, Distilled

> Free AI chat powered by GPT-4o. No API key. No sign-up. Always free.

---

## ⚡ Deploy to GitHub Pages (5 min)

> **Required:** App must run from HTTPS (GitHub Pages), not a local file.

### Step 1 — Create GitHub repo
1. Open **github.com** on your phone → tap **+** → **New repository**
2. Name: `lyra-ai` · Visibility: **Public** · tap **Create repository**

### Step 2 — Upload files
1. Tap **"uploading an existing file"**
2. Upload all 4 files: `index.html` · `manifest.json` · `sw.js` · `README.md`
3. Tap **Commit changes**

### Step 3 — Enable Pages
1. Tap **Settings** → scroll to **Pages**
2. Source: **Deploy from a branch** → Branch: **main** → folder: **/ (root)**
3. Tap **Save**

### Step 4 — Open Lyra
Visit: `https://YOUR-USERNAME.github.io/lyra-ai/`

---

## 📱 Install as Home Screen App
1. Open your GitHub Pages link in **Chrome** (Android)
2. Tap **⋮ → Add to Home Screen**
3. Launch Lyra like a native app!

## 🐛 Troubleshooting

| Problem | Fix |
|---------|-----|
| "Puter.js failed to load" | Open from GitHub Pages, not local file |
| AI not responding | Check internet · Try different model |
| Voice not working | Use Chrome · Allow mic permission |
| Blank screen | Hard refresh (Ctrl+Shift+R) |
